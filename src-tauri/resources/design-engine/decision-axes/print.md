# Print Decision Axes — 결정축 풀

## 축 카탈로그 (12개)

### 축 1: 인쇄물 종류
**무엇을 인쇄하나**
- 명함 (90×54mm 또는 89×51mm)
- 전단지/리플렛 (A4 또는 A3 접이식)
- 포스터 (A2/A1/B1)
- 카탈로그/브로셔 (A4 multi-page)
- 패키지 (박스·라벨·제품 표면)
- 사이니지 (간판·표지판 — 큰 사이즈)

### 축 2: 사이즈·비율
**최종 사이즈**
- 표준 (A4·A3·B5·명함 표준)
- 정사각 (소셜 공유용 + 인쇄 겸용)
- 와이드 (3:1, 5:1 — 배너·포스터)
- 커스텀

### 축 3: 컬러 방식
**인쇄 컬러 모델**
- 풀컬러 CMYK (4도 인쇄)
- 1도 (단색 — 비용 절감)
- 2~3도 (별색 사용 — Pantone 1~2개)
- 흑백 + 별색 1개 (모던·고급)

### 축 4: 정보 밀도
**한 면에 담는 정보량**
- Sparse (메시지 1개 강조 — 포스터)
- Balanced (적정 — 카탈로그 1페이지)
- Dense (제품 사양·표 — 데이터시트)

### 축 5: 시각 vs 텍스트 비중
**무엇이 주인공**
- 이미지 중심 (포토북·룩북)
- 타이포그래피 중심 (텍스트가 디자인)
- 일러스트 중심 (벡터 그래픽)
- 데이터 시각화 중심 (인포그래픽)

### 축 6: 사용 환경
**어디서 보나**
- 손에 들고 (명함·리플렛 — 30cm 거리)
- 벽에 (포스터·간판 — 1~5m 거리)
- 책상 위 (카탈로그·노트 — 60cm)
- 패키지/제품 위 (1~3m 매장 진열)

### 축 7: 종이 질감 가정
**디자인이 가정하는 인쇄 매체**
- 매끈 코팅지 (사진 우선)
- 무광 종이 (텍스트 가독)
- 거친 재생지 (친환경·따뜻함)
- 두꺼운 카드지 (명함·고급)

### 축 8: 텍스트 양
**카피 길이**
- 최소 (헤드라인 + 한 줄)
- 적정 (헤드라인 + 본문 1~2단)
- 풍부 (장문 — 카탈로그·매뉴얼)

### 축 9: 격식·톤
**인상**
- 공식적/엄숙 (학술·정부·금융)
- 비즈니스 (B2B 일반)
- 친근 (B2C·소비재)
- 아티스널 (수작업 느낌·갤러리)

### 축 10: 인쇄 후 가공
**다이컷·접지·후가공 가정**
- 단순 절단만 (저비용)
- 접지 (3단·4단 리플렛)
- 다이컷 (복잡한 모양)
- 박·엠보싱 (고급 후가공)

### 축 11: 메인 카피·헤드라인 강도
**시선 흐름**
- 거대 한 단어 (눈에 박힘)
- 짧은 문장 (3~5단어)
- 헤드라인 + 부제 (계단식)

### 축 12: 다국어 / 한국어 비중
- 한국어 only
- 한국어 메인 + 영문 보조
- 영문 메인 (수출용)
- 다국어 병기 (일·중·영)

## 도메인별 가중치

- **명함**: 축 1·2(표준 사이즈), 축 3(컬러 절제), 축 11(이름·직책 위계)
- **포스터**: 축 4(sparse), 축 5(이미지 또는 타이포 중심), 축 11(거대 헤드라인)
- **카탈로그**: 축 4(dense), 축 5(이미지+텍스트 균형), 축 8(풍부)
- **패키지**: 축 7(종이 질감), 축 10(후가공), 축 9(친근/고급)
- **브로셔**: 축 1(접이식), 축 8(적정), 축 4(balanced)
