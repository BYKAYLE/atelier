# App Decision Axes — 결정축 풀

## 축 카탈로그 (15개 — 우선순위 순)

### 축 1: 핵심 사용자 행동 (Primary Action)
**앱이 1번 하길 바라는 일**
- 정보 조회 (콘텐츠 소비 — 뉴스/SNS/문서)
- 작업 수행 (생산성 — todo/노트/금융)
- 거래 (구매·예약·결제)
- 모니터링 (대시보드·헬스케어·홈 IoT)

### 축 2: 사용 빈도·세션 길이
**얼마나 자주, 얼마나 길게**
- 짧고 잦음 (메신저·SNS — 1분 미만, 하루 다회)
- 중간·정기 (생산성·뉴스 — 5~20분, 하루 1~3회)
- 길고 가끔 (전문 도구·교육 — 30분+, 주 단위)
- 응급·필요시 (헬스케어·은행 — 5분, 월 단위)

### 축 3: 디바이스 우선순위
**어디서 가장 자주 쓰나**
- 모바일 only
- 모바일 + 태블릿
- 데스크탑 + 모바일 (반응형 웹앱)
- 모바일·웹·데스크탑 풀크로스

### 축 4: 화면 밀도 (Information Density)
**한 화면에 담는 정보량**
- Sparse (한 가지 작업에 집중 — Apple Notes, Things)
- Balanced (적정 — Slack, Notion)
- Dense (전문가용 — Bloomberg, IDE)

### 축 5: 인터랙션 깊이 (Interaction Depth)
**사용자 조작 복잡도**
- 단순 (탭·스크롤만)
- 중간 (드래그·스와이프·롱프레스)
- 풍부 (멀티터치·제스처·키보드 단축키)

### 축 6: 시각 톤 (Visual Tone)
**전체 무드**
- 라이트 모드 default (Apple iOS 시스템)
- 다크 모드 default (Linear, Discord)
- 컬러풀 (어린이·게임·라이프스타일)
- 모노+액센트 (전문가 도구)

### 축 7: 네비게이션 패턴 (Navigation)
**주요 화면 이동 방식**
- 하단 탭바 (3~5개 메인 섹션 — iOS 표준)
- 사이드바 (데스크탑 우선·다층 구조)
- 단일 흐름 (linear — 온보딩·체크아웃)
- 탭 + 카드 스택 (혼합)

### 축 8: 카드 vs 리스트 (Content Layout)
**콘텐츠 표시 방식**
- 카드 그리드 (이미지 중심 — Pinterest, 쇼핑)
- 행 리스트 (텍스트 중심 — 메신저, 메일)
- 혼합 (피드 형태 — Twitter, Instagram)

### 축 9: 모션·애니메이션 강도
**전환·피드백 톤**
- 정적 (모션 최소 — 빠른 응답 우선)
- 미세 (탭 피드백·전환 200ms — iOS 네이티브 톤)
- 풍부 (스프링 애니·페이지 전환 강조 — 게임·라이프스타일)

### 축 10: 카피 톤 (Copy Voice)
**문장 톤**
- 격식 (~합니다 / 금융·헬스케어)
- 비즈니스 (~합니다 / 생산성 도구)
- 캐주얼 (~해요 / SNS·라이프스타일)
- 짧고 동사형 (메뉴 라벨 위주)

### 축 11: 타깃 사용자 IT 숙련도
**기술 친숙도**
- 시니어/비숙련 (큰 글자·아이콘 명확·단순)
- 일반 (표준 패턴 따름)
- 파워 유저 (단축키·제스처·고밀도)

### 축 12: 데이터 표시 우선순위
**핵심 정보 형태**
- 텍스트·문서 중심
- 숫자·차트 중심 (대시보드)
- 이미지·미디어 중심
- 폼·입력 중심 (설문·예약)

### 축 13: 온보딩 깊이
**첫 사용 안내 강도**
- 거의 없음 (자명한 인터페이스)
- 가벼운 투어 (3 스크린 슬라이드)
- 단계별 가이드 (튜토리얼 + 빈 상태 안내)
- 깊은 설정 (계정·환경설정 진입 강제)

### 축 14: 알림·피드백 강도
**상태·결과 표시**
- 조용함 (성공 시 무반응 — 부드러운 가이드)
- 토스트·인라인 (적정)
- 강한 피드백 (모달·진동·소리)

### 축 15: 접근성 우선도
**A11y 강도**
- 표준 (WCAG AA 기본)
- 강화 (큰 글자·고대비·VoiceOver 최적화 — 헬스케어·시니어 타겟)
- 다국어 우선 (RTL·다언어 토글)

## 도메인별 가중치

- **헬스케어 앱**: 축 1(작업/모니터링), 축 11(숙련도), 축 14(피드백 강함), 축 15(접근성 강화) 우선
- **금융 앱**: 축 1(거래), 축 9(정적·신뢰), 축 14(피드백 명확)
- **소셜/메신저**: 축 2(짧고 잦음), 축 8(혼합 피드), 축 9(풍부)
- **생산성 도구**: 축 4(밀도), 축 7(사이드바), 축 11(파워 유저)
- **게임/라이프스타일**: 축 6(컬러풀/다크), 축 9(풍부), 축 13(온보딩 깊음)
