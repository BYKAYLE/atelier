# CI Decision Axes — 결정축 풀

질문 생성 LLM은 brief를 분석하고, 아래 풀에서 **brief에 명시되지 않은 + 출력에 결정적 영향을 주는** 축 4~5개를 골라 사용자에게 묻는다.

각 축은 양극성(spectrum) 또는 다중 선택. 사용자가 답하기 쉬운 일상 언어로 표현.

## 축 카탈로그 (20개 — 우선순위 순)

### 축 1: 시간성 (Temporal Strategy)
**브랜드를 언제까지 쓸 건가**
- 5년+ 변하지 않을 timeless 정체성 (안정·신뢰 우선)
- 회사 성장 단계마다 진화 가능한 adaptive 유연성 (실험·확장 우선)
- 균형 (둘 다 중요)
**질문 영향**: 로고 추상도, 타이포 클래식도, 컬러 채도

### 축 2: 무게감 (Visual Weight)
**첫 인상의 무게**
- 신뢰의 무게 (전통·클래식·기관 톤)
- 혁신의 가벼움 (미래·실험·스타트업 톤)
- 중립
**질문 영향**: 폰트 weight, 색 채도, 형태 곡률

### 축 3: 추상도 (Abstraction Level)
**로고가 무엇을 형상화하나**
- 구상적 (실물 메타포 — 손/도구/사물)
- 추상적 (geometric / 의미를 단순화)
- 타입 중심 (워드마크만, 심볼 없음)
**질문 영향**: 로고 형태, 스토리텔링 깊이

### 축 4: 차별 전략 (Differentiation)
**산업 컨벤션에 대한 태도**
- 산업 코드 활용 (의료=십자가/모니터, F&B=음식/포크)
- 의도적으로 깨기 (예측 안 되는 메타포)
- 인접 산업 코드 차용 (의료에 IT 톤 가져오기)
**질문 영향**: 메타포 선택, 시장 인지 비용

### 축 5: 인지 친밀도 (Familiarity)
**타겟이 받는 인상**
- 익숙한 안전감 (이미 본 듯한 안정감)
- 도전적 새로움 (낯설지만 끌림)
- 균형
**질문 영향**: 컬러 톤, 형태 친숙도

### 축 6: 확장 우선순위 (Application Priority)
**가장 자주 노출될 매체**
- 디지털 우선 (앱 아이콘·웹 헤더·SNS)
- 인쇄 우선 (명함·패키지·사이니지)
- 공간/실물 우선 (간판·전시·유니폼)
- 영상 우선 (오프닝·BI 인트로)
**질문 영향**: 로고 viewBox 비율, 최소 크기, 모션 적용도

### 축 7: 시각 시그니처 위치 (Signature Locus)
**브랜드 자산의 무게중심**
- 로고 중심 (강한 brand mark가 모든 응용 주도)
- 시스템 중심 (컬러·타이포·패턴 조합이 정체성)
- 사진/이미지 중심 (사진 톤이 브랜드)
**질문 영향**: 로고 정밀도 vs 시스템 폭 vs 사진 가이드 비중

### 축 8: 컬러 강도 (Color Intensity)
**컬러의 역할**
- 강한 단색 (1색 강력 — 코카콜라 빨강 톤)
- 부드러운 팔레트 (2~3색 조화)
- 흑백+액센트 (모노톤 + 1포인트)
- 다채로운 (4색 이상 — 어린이·F&B 톤)
**질문 영향**: 컬러 시스템 구조, 응용 일관성 난이도

### 축 9: 형태 언어 (Form Vocabulary)
**라인의 성격**
- 직선/각진 (geometric — 정확·기술)
- 곡선/유기적 (인간적·따뜻함)
- 손글씨/calligraphic (수공·아티스널)
- 혼합
**질문 영향**: 로고 곡률, 그리드, 모서리 처리

### 축 10: 격식 톤 (Formality)
**커뮤니케이션 격식**
- 공식적/엄숙 (학술·정부·금융)
- 비즈니스 캐주얼 (B2B 일반)
- 친근/친밀 (B2C·라이프스타일)
- 유머러스 (의외성)
**질문 영향**: 타이포, 카피 스타일

### 축 11: 한국어/영문 비중
**브랜드명 표기 우선순위**
- 한글 메인 + 영문 보조
- 영문 메인 + 한글 보조
- 둘 다 동등 (병기)
- 영문만
**질문 영향**: 워드마크 디자인, 자간 조정

### 축 12: 글로벌/로컬 (Geographic Scope)
**사용 지역**
- 한국 시장 한정 (국내 컨벤션 우선)
- 글로벌 적용 (문화 중립·번역 가능)
- 한국+일본+동남아 (아시아 컨벤션)
**질문 영향**: 색 의미, 메타포 선택, 폰트 다국어 지원

### 축 13: 타깃 세대 (Generation)
**핵심 타겟**
- 시니어 전문가 (40~60대)
- 비즈니스 일반 (30~50대)
- MZ/청년 (20~30대)
- 어린이/학부모
**질문 영향**: 톤, 컬러, 친밀도

### 축 14: 모회사·서브브랜드 관계 (Brand Hierarchy)
**기존 자산 활용 여부**
- 독립 신생 (제로부터)
- 모회사 자산 활용 (서브브랜드 — 통일성)
- 자매 브랜드 시리즈 (시리즈 일관성 + 변형)
- 리브랜딩 (기존 인지도 일부 보존)
**질문 영향**: 시각 자유도, 차별 vs 통일

### 축 15: 시각적 무게중심 (Composition Balance)
**구성 톤**
- 대칭 (안정·고전)
- 비대칭 (긴장·현대)
- 중앙 집중 (집중·존엄)
**질문 영향**: 로고 컴포지션, 응용 레이아웃

### 축 16: 모션 적용도 (Motion)
**브랜드의 모션 비중**
- 정적 (모션 없음 — 인쇄 중심)
- 미세 모션 (호버·로딩 정도)
- 모션 그래픽 강함 (영상 인트로·BI 모션)
**질문 영향**: 로고 변형 가능성, BI 가이드 motion 섹션 비중

### 축 17: 정보 밀도 (Density)
**한 응용 매체에 담는 정보량**
- 미니멀 (로고 + 한 줄)
- 중간 (로고 + 슬로건 + 연락처)
- 풍부 (로고 + 시스템 다종 동시 노출)
**질문 영향**: 명함·레터헤드 디자인, 응용 mockup 복잡도

### 축 18: 메타포 우선순위 (Meaning Strategy)
**상징성 vs 시각적 즐거움**
- 깊은 의미·상징 강조 (스토리텔링)
- 순수 시각 (깔끔한 형태가 자체 가치)
- 균형
**질문 영향**: 로고 컨셉 깊이, 스토리북 작성 필요 여부

### 축 19: 시각적 강도 (Impact)
**눈에 띄는 정도**
- 조용함 (오래 보아야 좋음)
- 임팩트 (한 번에 강한 인상)
- 균형
**질문 영향**: 컬러 강도, 컴포지션, 폰트 weight

### 축 20: 산업/도메인 컨벤션 따름 정도
**산업 표준 가이드라인 준수**
- 강하게 따름 (의료=청결, 금융=신뢰 색)
- 부드럽게 차용 (산업 톤 + 차별)
- 완전 깸 (의외성으로 차별)
**질문 영향**: 시장 인지 비용, 차별성

## 질문 생성 LLM 사용 가이드

1. **brief를 먼저 분석** — brief에 이미 명시된 축은 절대 묻지 말 것
   - 예: brief에 "의료진이 신뢰하는 클래식한 톤"이라고 했으면 축 2 무게감, 축 5 친밀도, 축 10 격식 톤 모두 답이 있음 → 묻지 않음
2. **결정적 5축 선택** — 답에 따라 출력이 가장 많이 달라지는 축 우선
3. **질문 표현은 일상 언어** — "추상도"라고 묻지 말고 "로고가 무엇처럼 보이게 하시겠어요? (실물 메타포 / 추상 도형 / 글자만)"
4. **각 질문에 "Decide for me" 옵션 포함** — 사용자가 결정 못 해도 진행 가능
5. **2개 양극 + "균형" 옵션 + "Decide for me"** 형태가 답하기 가장 쉬움
6. **자유 텍스트 질문 1~2개 추가** — "피하고 싶은 표현", "꼭 들어가야 할 키워드"
7. **5±2 한도** — 7개 넘으면 사용자가 답 안 함

## 도메인별 가중치 가이드

- **의료/제약/헬스케어**: 축 2(무게감), 축 4(차별), 축 10(격식), 축 20(산업 컨벤션) 우선
- **F&B/리테일/라이프스타일**: 축 8(컬러), 축 10(격식), 축 13(세대), 축 17(밀도) 우선
- **교육/학술**: 축 1(시간성), 축 10(격식), 축 11(언어), 축 13(세대) 우선
- **테크/SaaS/스타트업**: 축 1(시간성), 축 6(확장), 축 14(브랜드 hierarchy), 축 16(모션) 우선
- **금융/공공**: 축 2(무게감), 축 4(차별), 축 10(격식), 축 19(임팩트 — 조용함 우선) 우선
