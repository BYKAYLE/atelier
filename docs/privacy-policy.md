# Atelier Privacy Policy

Atelier is a desktop application for running local command-line coding agents,
terminal sessions, image paste workflows, and local preview workflows.

## Data Collection

Atelier does not operate a cloud service and does not intentionally collect,
sell, rent, or share personal data.

The app stores local settings and workspace metadata on the user's device so it
can restore profiles, sessions, preferences, and preview configuration.

## Local Tools and Network Access

Atelier can launch command-line tools selected by the user, such as coding
agents, shells, package managers, or local development servers. Those tools may
connect to their own services depending on the user's configuration,
credentials, and the tools' separate terms and privacy policies.

Atelier does not send those credentials or terminal contents to a BYKAYLE-owned
server.

## Clipboard and Images

When the user pastes an image, Atelier may save a temporary local copy so the
selected terminal or agent can use it. These temporary files remain on the
user's device unless the user or a launched tool chooses to send them elsewhere.

## Updates

Atelier may check a configured release endpoint to determine whether an update
is available. The update check is limited to standard release metadata.

## Contact
